Detailed Description

Logger Manager for the simple logging package. The role of the LoggerManager is to record loggers (

See also
Logger). By default a "root logger" (which is a console logger) is created. When using a logger, end line to display information by sending the '
' character or Logger::endl.

How to use simple logging system:

  • register new ConsoleLogger called stdout: 
    
LoggerManager::get_instance().attach(new ConsoleLogger("stdout", &std::cout));
  • use logger stdout: 
    
Logger& log1 = LoggerManager::instance().get_logger("stdout");
log1 << "hello world\n";
log1 << "message" << Logger::endl;
  • register new FileLogger called my: 
    
LoggerManager::instance().attach(new FileLogger("my", "file_logger.txt",
FileLogger::TRUNCATE));
    Logger& log2 = LoggerManager::get_instance().get_logger("my");
