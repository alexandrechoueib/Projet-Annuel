◆ is_integer()

bool Types::is_integer ( std::string &  s)
static

Check whether string is 32 bits integer. The integer can be given in binary ('b' suffix), octal ('o' suffix), hexadecimal ('h' suffix) or in decimal ('d' suffix or no suffix). The underline '_' character can be used to improve the reading of the integer. Some valid examples are:

  • -123 or -123_d or -123d
  • ffh or ff_h or FF_h which corresponds to 255 in decimal
  • 1_0001_0001b or 1_0001_0001_b or 100010001b which corresponds to 273 in decimal
