Detailed Description

template<class DataType>
class ez::objects::Array< DataType >

Generic class that represents an array of DataType values. The indexes of the array are taken in the RangeType interval which has integer type. For example we can represent an array of real numbers with indexes that range from -3 to 10 by the following declaration: Array<real> a(-3,10)
