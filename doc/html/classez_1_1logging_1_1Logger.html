Detailed Description

base class for loggers which is composed of:

  • a name which serves as an identifier to retrieve the logger from the LoggerManager
  • a pointer to an output stream

The Logger needs to be attached to the LoggerManager (

See also
LoggerManager) Each line displayed must be ended by '
' or by using << Logger::endl. Use << setlevel(int) to fix the level of information recorded.
